OBS Setup
How to Stream with OBS Studio
  1. Open OBS Studio
  2. Go to Settings > Stream
  3. Select "Custom..." as the service
  4. Server URL: rtmp://{{ request.host }}/live
  5. Stream Key: Your stream key
  6. Click "Apply" then "OK"
  7. Start streaming in OBS

Recommended Output Settings:

  • Output Mode: Simple
  • Video Bitrate: Match your stream settings
  • Encoder: x264 (or Hardware if available)
  • Audio Bitrate: 128Kbps
  • Enable Advanced Encoder Settings: Checked
  • Preset: veryfast
